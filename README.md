## Speed 

### 1 million lines =~ 2.252 seconds to parse and save into a variable


Benchmarks were measured under the following conditions:

- 1 line creating a variable (`.new`)
- 999,999 lines editing the same variable (`.edit`)
- PVN version: Beta 0.0.4
- Ruby: 3.4.8
- CPU: AMD Ryzen 5 7600X
- Memory: 32 GB DDR5
